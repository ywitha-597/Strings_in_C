Flashcards - C Programming

strcpy

Description: Copies the string pointed to by source (including the null character) to the destination array.
Parameters: char* strcpy(char* destination, const char* source);
Trick: Think of `strcpy` as copying the content of one string to another. The destination (`destination`) is where you want to copy, and the source (`source`) is where the content comes from.

strcat

Description: Appends a copy of the source string to the destination string. The terminating null character in destination is overwritten by the first character of source, and a null-character is included at the end of the new string formed by the concatenation of both in destination.
Parameters: char* strcat(char* dest, const char* src);
Trick: Imagine a cat (CAT) joining two strings together. The destination string (`dest`) is like the cat's home, and the source string (`src`) is like the cat wandering around. The cat joins the two strings together.

strcmp

Description: Compares the string pointed to by str1 to the string pointed to by str2.
Parameters: int strcmp(const char *str1, const char *str2);
Trick: When you compare two strings (`str1` and `str2`) using `strcmp`, it returns an integer indicating whether the strings are equal, less than, or greater than each other. Think of it as comparing two strings for equality.

strlen

Description: Returns the length of the string pointed to by str, excluding the null character.
Parameters: int strlen(const char *str);
Trick: With `strlen`, you're finding the length of a string (`str`). It's like measuring the length of a rope. You pass the string to `strlen`, and it tells you how long the string is.
